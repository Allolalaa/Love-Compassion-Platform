## 🎥 إضافة فيديو جديد

### الخطوات:

1. **افتح ملف** `src/components/VideoLibrary.tsx`

2. **ابحث عن** قسم `const videos = [`

3. **أضف فيديو جديد** بنسخ هذا النموذج:
   ```javascript
   {
     id: 7, // رقم تسلسلي جديد
     title: 'عنوان الفيديو الجديد',
     category: 'talks', // اختر من: talks, interviews, tutorials, shorts
     thumbnail: 'رابط الصورة المصغرة',
     duration: '15:30', // مدة الفيديو
     views: '0',
     date: 'منذ يوم',
   },
   ```

4. **احفظ الملف**

### ملاحظات مهمة:
- تأكد من استخدام رقم ID فريد لكل فيديو
- التصنيفات المتاحة: `talks` (محاضرات)، `interviews` (مقابلات)، `tutorials` (دروس)، `shorts` (أفلام قصيرة)

## 📖 إنشاء دورة تدريبية جديدة

### الخطوات:

1. **افتح ملف** `src/components/LearningHub.tsx`

2. **ابحث عن** قسم `const courses = [`

3. **أضف دورة جديدة**:
   ```javascript
   {
     id: 5,
     title: 'عنوان الدورة',
     description: 'وصف تفصيلي للدورة',
     level: 'مبتدئ', // أو: متوسط، متقدم
     duration: '4 أسابيع',
     lessons: 12,
     students: 0,
     rating: 5.0,
     image: 'رابط صورة الدورة',
     modules: [
       'المحور الأول',
       'المحور الثاني',
       'المحور الثالث',
       'المحور الرابع',
     ],
   },
   ```

4. **احفظ الملف**

### نصائح:
- استخدم صور عالية الجودة (على الأقل 800x450 بكسل)
- اكتب وصف واضح وجذاب للدورة
- رتب المحاور بشكل منطقي من البسيط للمعقد

## 💬 إدارة المنتدى

### إضافة موضوع مثبت:

1. **افتح ملف** `src/components/Forum.tsx`

2. **ابحث عن** قسم `const topics = [`

3. **أضف موضوع جديد**:
   ```javascript
   {
     id: 6,
     title: 'عنوان الموضوع',
     author: 'اسم الكاتب',
     avatar: '👤', // اختر إيموجي مناسب
     category: 'نقاشات عامة', // أو أي تصنيف آخر
     replies: 0,
     views: 0,
     lastActivity: 'منذ دقائق',
     isPinned: true, // true للموضوع المثبت، false للعادي
     excerpt: 'مقتطف من الموضوع...',
   },
   ```

### إدارة التصنيفات:

- **نقاشات عامة**: للمواضيع العامة
- **الدروس والمحاضرات**: للنقاشات حول المحتوى التعليمي
- **المبادرات والمشاريع**: للأفكار والمشاريع
- **قصص النجاح**: لمشاركة التجارب الملهمة

## 🎤 جدولة ندوة مباشرة

### الخطوات:

1. **افتح ملف** `src/components/Events.tsx`

2. **ابحث عن** قسم `const upcomingEvents = [`

3. **أضف فعالية جديدة**:
   ```javascript
   {
     id: 4,
     title: 'عنوان الندوة',
     date: '25 مارس 2024',
     time: '19:00 - 21:00',
     type: 'ندوة مباشرة', // أو: ورشة تفاعلية، جلسة حوارية
     speaker: 'اسم المتحدث',
     participants: 0,
     maxParticipants: 500,
     image: 'رابط صورة الفعالية',
     description: 'وصف الفعالية',
     isLive: false, // true للفعاليات المباشرة الآن
   },
   ```

### إضافة تسجيل سابق:

بعد انتهاء الندوة، انقلها إلى `const pastEvents = [`

```javascript
{

  date: '20 مارس 2024',
  views: 0,
  duration: '1:45:00',
  image: 'رابط الصورة',
  recordingUrl: 'رابط التسجيل',
```

## 🎓 مصطلحات تقنية بسيطة

- **Component**: مكون - ملف يحتوي على جزء من الموقع
- **Array**: مصفوفة - قائمة بالعناصر `[...]`
- **Object**: كائن - مجموعة من البيانات `{...}`
- **ID**: معرف فريد - رقم لا يتكرر
