## ✨ الميزات الرئيسية

### 📚 محتوى تعليمي غني

#### 1. **مكتبة الفيديو**
- مكتبة شاملة للأفلام القصيرة والمقابلات
- نظام تصنيف متقدم (محاضرات، مقابلات، دروس، أفلام قصيرة)
- عداد للمشاهدات والتاريخ
- واجهة مستخدم جذابة مع معاينة تفاعلية

#### 2. **معرض الرسوم البيانية**
- معرض جذاب للرسوم البيانية التوضيحية
- إمكانية التحميل والمشاركة
- نظام تصنيف (إحصائيات، إرشادات، خرائط، تطوير ذاتي)
- عرض مكبر للتفاصيل

#### 3. **مركز التعلم الإلكتروني**
- دورات تدريبية شاملة قابلة للتوسع
- نظام مستويات (مبتدئ، متوسط، متقدم)
- عرض تفصيلي للمحاور والدروس
- تقييمات ومراجعات
- مسار تعليمي موصى به

### 🎯 تجربة تفاعلية

#### 4. **منتدى النقاش**
- نظام منتدى متكامل للنقاشات
- تصنيفات متعددة للمواضيع
- إمكانية تثبيت المواضيع المهمة
- عداد للردود والمشاهدات
- واجهة نظيفة وسهلة الاستخدام

#### 5. **قسم الفعاليات**
- جدولة الندوات المباشرة
- نظام التسجيل والحد الأقصى للمشاركين
- أرشيف للتسجيلات السابقة
- عداد المشاهدات للتسجيلات
- إشعارات الفعاليات القادمة

## 📦 التثبيت والتشغيل

### المتطلبات:
- Node.js (الإصدار 16 أو أحدث)
- npm أو yarn

### خطوات التثبيت:

1. **استنساخ المشروع**
   ```bash
   git clone [repository-url]
   cd love-compassion-platform
   ```

2. **تثبيت الحزم**
   ```bash
   npm install
   ```

3. **تشغيل السيرفر المحلي**
   ```bash
   npm run dev
   ```

4. **بناء للإنتاج**
   ```bash
   npm run build
   ```

5. **معاينة البناء**
   ```bash
   npm run preview
   ```
