## ✨ الميزات الرئيسية

### 📚 محتوى تعليمي غني
1. **مكتبة الفيديو:** نظام متقدم لتصنيف الأفلام والمقابلات مع عداد للمشاهدات ومعاينة تفاعلية.
2. **معرض الرسوم البيانية:** عرض جذاب للبيانات والإحصائيات مع إمكانية التحميل المباشر.
3. **مركز التعلم الإلكتروني:** دورات تدريبية مقسمة لمستويات (مبتدئ، متوسط، متقدم) مع تتبع المحاور.

### 🎯 تجربة تفاعلية
4. **منتدى النقاش:** نظام متكامل لتبادل الأفكار وتثبيت المواضيع المهمة.
5. **قسم الفعاليات:** نظام لجدولة الندوات المباشرة، التسجيل، وأرشفة اللقاءات السابقة.

## 🛠️ البنية التقنية (Tech Stack)

تم بناء المنصة باستخدام أحدث التقنيات لضمان أداء فائق وسرعة تحميل استثنائية:
- **React 19**: لإدارة الواجهات البرمجية وتوفير تجربة مستخدم سلسة.
- **TypeScript**: لضمان كود آمن، منظم، وقابل للتوسع.
- **Tailwind CSS**: لتصميم عصري، متجاوب (Responsive)، ودعم كامل للـ RTL.
- **Vite**: أداة البناء الحديثة لضمان سرعة التطوير والتشغيل.
- **Vite Single File**: لتحسين أداء تحميل الصفحة الواحدة (Single Page Application).
